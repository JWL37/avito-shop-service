# avito-shop-service

## Запуск сервиса 

Для запуска нужен докер. В директории avito-shop-service нужно выполнить:

```shell
docker-compose up -d
```

## Линтер

Для запуска нужно скачать golangci-lint.

Запускать можно через утилиту make:

```shell
make lint
```

или запустить скрипт вручную

```shell
./run_lint.sh
```
